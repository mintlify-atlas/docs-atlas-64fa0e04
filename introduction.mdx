## What is Express?

Express is a fast, unopinionated, minimalist web framework for [Node.js](https://nodejs.org). It provides a robust set of features for building web and mobile applications, making it one of the most popular Node.js frameworks.

<Note>
Express requires Node.js 18 or higher. Make sure you have Node.js installed before getting started.
</Note>

## Philosophy

The Express philosophy is to provide small, robust tooling for HTTP servers, making it a great solution for:

- Single page applications
- Traditional websites
- Hybrid applications
- Public HTTP APIs
- Microservices

Express does not force you to use any specific ORM or template engine. With support for over 14 template engines via [@ladjs/consolidate](https://github.com/ladjs/consolidate), you can quickly craft your perfect framework.

## Quick Example

Here's a simple Express server that responds with "Hello World":

```javascript
import express from 'express'

const app = express()

app.get('/', (req, res) => {
  res.send('Hello World')
})

app.listen(3000, () => {
  console.log('Server is running on http://localhost:3000')
})
```
